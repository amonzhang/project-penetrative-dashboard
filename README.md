# 穿透式工程项目管理图形化建模与价值—资金一体化驾驶舱

> **面向复杂工程项目的“价值—资金一体化”穿透式可视化决策系统。**
> 本仓库包含一整套从**学术方法论设计**、**纯 HTML 本地轻量演练沙箱**，到基于 **Next.js 16 + React 19 + TypeScript + 3D Capsule 极简图形引擎** 构建的交互式大屏决策驾驶舱原型。

---

## 📖 项目背景与管理理念

在大型、复杂且高动态的工程项目中，**进度（甘特图）**、**成本（财务口径）**与**现金流（收付实现）**往往由不同的部门在不同报表中独立维护。由于口径、节拍割裂，管理者和业主极难直观、动态地判断项目真实利润与融资垫资缺口。

本项目践行**“穿透式管理”**的方法论，创新性地实现“计划价值（PV）—挣得价值（EV）—实际成本（AC）—现金收付（In/Out/Net）”的**统一时间轴对齐**与**穿透式审计下钻**：

*   **统一标尺对齐**：以计划合同总额（=100%）作为同轴标尺。
*   **三行物理对齐**：预算控制导轨、实际成本胶囊、流动资金条在同一视域物理层层对齐，偏差一目了然。
*   **穿透式瀑布挖掘**：
    *   **Track A（收入与回款）**：合同总盘 $\rightarrow$ 产值确权瀑布 $\rightarrow$ 单笔收款凭证。
    *   **Track B（内部经营与成本）**：成本大类分类 $\rightarrow$ 对应供应商预算占比 $\rightarrow$ 供应商付款凭证。
*   **适配不同层级**：同屏支持“只看偏差（新手模式）”与“全项穿透（专家模式）”。

## 🎨 3D Minimalist Graphic Engine (三维极简图形引擎)

为了在浅色大屏模式下实现精致的物理触感与机械质感，我们于子模块 `dashboard/` 中自主设计并封装了 **3D 极简图形引擎 (`PenetrativeGraphicEngine.tsx`)**。核心亮点包括：

1.  **物理雕刻凹槽导轨 (`BudgetSlot`)**：使用 `rounded-full` 圆角与三层内阴影（`shadow-[inset_0_3px_5px]`），配合亮白 bevel 边框，在页面上雕刻出高度逼真的**物理切削下凹轨道**。
2.  **3D 半透明玻璃滑柱 (`CylinderCapsule`)**：两端采用球形胶囊过渡。内置圆柱面渐变滤镜 `CYLINDER_REFLECTION` 与反射亮丝（`bg-white/45 blur-[0.5px]`），逼真还原**玻璃圆柱滑块**在日光下的漫反射和曲率。
3.  **斜纹液态流动 3D 条 (`HatchedFlow`)**：对现金流斜线层注入 CSS `flowStripes` 动画，展现**资金在管道内向右流动的滑润微动画**；同时在其最右侧边缘设有呼吸发光的**激光端点指示灯**。
4.  **工业管路直角接线器 (`TreePipeConnector`)**：采用直角反光金属管道连线，并在直角折角处镶嵌了一枚**微光 LED 状态指示圆珠**，强化穿透折叠时的“通电导通感”。

## ⚡ 快速开始与使用指南

## ⚙️ 技术与构建规范
*   **水合冲突防御 (SSR Safe)**：核心组件通过 `dynamic(..., { ssr: false })` 打包，彻底根除了 React 19 因服务端随机数据导致的 `Hydration Failed` 死穴，交互零卡顿。
*   **浅色画卷美学 (Light Mode)**：以优雅柔和的 `slate-50` 暖白作画卷，白卡片、浅灰字、高对比彩色柱状图，色彩层次高级温润。
*   **静态验证通过**：通过 `npx tsc --noEmit` 类型校验，且 `npm run build` 生产构建 **0 警告，0 报错**。
