# 三行对齐进度条 + S曲线 + 现金流 + 时间滑块

- 1. 锚点视图：三行对齐进度条
  - 作用：作为直观锚点，快速对齐预算、实际、现金流
  - 三行内容
    - 预算行：计划收入→成本分项→利润
    - 实际行：已实现收入→已发生成本分项→实际利润
    - 现金流行：收款→付款分项→净现金
  - 特点：左端对齐，一眼看出偏差

- 2. 上方 S 曲线（价值进度）
  - 核心要素：PV、EV、AC
  - 指标：CPI、SPI
  - 意义：显示效率与进度，是专业PM主要工具

- 3. 下方现金流曲线
  - 三条线：累计收款、累计付款、累计净现金
  - 意义：判断资金缺口、融资需求
  - 结合 S 曲线：对比价值产出与资金状况

- 4. 时间滑块与交互
  - 功能：拖动时间→曲线动态生长
  - 交互方式：只看偏差/新手模式/专家模式

- 5. 管理意义总结
  - 认知减负：多层次视图降低理解门槛
  - 多维观察：进度、成本、现金流三维一体
  - 适配不同层级：新手、PM、管理层/业主
