# 穿透式工程项目管理驾驶舱 (Penetrative Project Management Dashboard)

> **基于“价值—资金一体化”管理理念构建的极简 3D 穿透式项目管理决策驾驶舱原型。**
> 本项目采用 **Next.js 16 (Webpack 模式) + React 19 + Tailwind CSS + TypeScript** 架构构建，将传统的挣值管理（EVM）与实际资金收付口径在同一视域内进行极致对齐与穿透式瀑布下钻。

---

## 🌟 管理理念与核心价值

在传统工程项目管理中，**进度计划（挣值 EVM）**、**实际发生成本（费用口径）**与**实际现金收付（收付实现制）**往往分处不同报表，口径割裂且时相不一。这导致管理者在进行财务审计与变更索赔时，需要在脑内完成复杂的“跨图对齐”，决策门槛极高。

本项目贯彻**“穿透式管理”**方法论，创新性地提出：
1. **统一视域标尺**：以计划合同收入（=100%）作为全局统一坐标轴。
2. **三行同轴对齐**：预算基线、实际挣值、现金流收付三线物理对齐，一眼识别盈亏与融资窗口。
3. **“穿透”下钻审计**：支持从合同总额、大类分类、供应商预算占比，一直穿透挖掘至单笔最底层的实际收款/付款凭证，实现“见图知账，见账知单”。

## 🎨 3D Minimalist Graphic Engine (三维极简图形引擎)

为了使穿透对比关系更具视觉冲击力与触觉质感，系统建立了一套自主设计的 **3D 极简图形引擎 (`PenetrativeGraphicEngine.tsx`)**。它利用纯 Vanilla CSS / Tailwind 配合物理光学折射公式，在 2D DOM 上再现了晶莹透亮的 3D 滑动件质感：

### 1. 物理雕刻凹槽导轨 (`BudgetSlot`)
* **凹陷控制卡槽**：采用双层内阴影（`shadow-[inset_0_3px_5px_rgba(15,23,42,0.08)]`）和底部白色 Bevel 高光边缘，模拟在实体控制面板上**物理切削切槽的下陷立体质感**。
* **中央定位导轨**：卡槽中轴贯穿了一根微透明 dashed 准星线，极具精准滑轨的精密感。

### 2. 3D 半透明玻璃滑柱 (`CylinderCapsule`)
* **圆柱镜面折射**：胶囊两端采用 `rounded-full` 圆柱帽。覆盖多段圆柱渐变遮罩 `CYLINDER_REFLECTION`（顶部 specularity 亮条、中部高透光、底部深折射阴影），模拟高反光圆柱体的物理曲率。
* **高反光环境亮条**：胶囊顶部内置一根 `bg-white/45 blur-[0.5px]` 的极细反射亮丝，令 3D 胶囊在视觉上温润剔透。
* **悬停立体升维**：当鼠标划过时，滑杆自动纵向拉伸（`scale-y-[1.08]`），投影深度爆发，跃然屏上。

### 3. 斜纹液态流动 3D 嵌套条 (`HatchedFlow`)
* **无限流速微动画**：现金流/收付款的斜线阴影部分注入了 CSS Keyframe 动画（`flowStripes`），呈现出**无限向右平移的液态流动效果**，代表着资金正在管道内缓缓输送。
* **荧光激光边界端点**：流动条的最右前沿，设有一根呼吸发光（`animate-pulse-glow`）的荧光激光边界亮点，把实际资金进度标示得格外亮眼。

### 4. 工业管路折线分接器 (`TreePipeConnector`)
* **直角机械穿透树**：抛弃了单调的虚线树，代之以精致细密的**直角 3D 管道连线**，具备金属反光质感。
* **发光 LED 连接节点珠**：在管线每一次折角或分接点上，均镶嵌了一枚由主轨状态色（如 Track A 蓝色、Track B 分类色）染色的**微光 LED 节点圆珠**，带来逻辑通电导通的极致科技美。

## 📊 核心业务视图与交互细节

### Track A · 收入与回款（单轨同轴 merge）
业主视角的**产出确权**（EVM 中的 EV）与**收款资金流**合并到同一根 3D 导轨中呈现。
* **三层树形瀑布穿透**：
  $$\text{合同总盘} \longrightarrow \text{确权瀑布明细} \longrightarrow \text{关联单笔收款凭证}$$
* 支持确权实框对收款阴影液体的物理包裹，底层的每笔收款也利用比例反推确权实框，达成一致的对比逻辑。

### Track B · 内部经营与成本分类穿透
项目内部**实际发生成本**与**实付资金流**在各成本维度下的穿透控制。
* **三层树形瀑布穿透**：
  $$\text{大类分项 (材料/分包/其他/利润)} \longrightarrow \text{各供应商预算占比} \longrightarrow \text{供应商单笔付款凭证}$$
* **供应商瀑布对齐**：支持两种横向对齐与排序方式：
  * **按预算占比**：从大到小横向对齐摆放，一眼看清材料/分包的主力商。
  * **按首次确权时间**：从左往右按供应商第一次产值确权发生的先后顺序瀑布向下排列，精准判断进场节奏。

## ⚙️ 系统工程特性

### 1. 100% 修复 SSR 水合冲突 (Hydration Mismatch Fix)
* **技术难点**：在 React 19 + Next.js 环境下，因为仿真引擎随机生成的财务流水数据在 Server 端与 Client 端渲染时产生微弱噪声冲突，容易触发 React 的水合崩溃（Hydration Mismatch），导致 React 所有定时器、折叠交互和浮动条逻辑全部锁死。
* **极致方案**：在 `app/page.tsx` 入口处对核心驾驶舱实施 **SSR 动态打包隔离** (`dynamic(..., { ssr: false })`)，彻底消除水合死穴，保证在各种网络状况下页面闪电级加载，交互零卡顿。

### 2. 极简浅色画卷模式 (Light Mode 画卷)
* 告别压抑的暗色系，以极简清爽的高对比 slate-50/暖白（`#f8fafc`）作为主画布，纯白轻盈卡片配合三重微弥散阴影层，完美烘托出彩色极简 3D 胶囊的温润与光泽。

### 3. 随动浮动控制条 (Sticky Slider Bar)
* 演练控制面板被设计为 `sticky top-4 z-40` 的高档毛玻璃玻片。随着页面的向下滚动，演练进度条会轻盈地滑行并吸附在视口顶部，极大地方便了管理者的演练点按。

### 4. 浮动玻片 Micro-Tooltip 与发光药丸
* 鼠标移动到 Bar 上时，纯白高光、双层边框的浮动玻片 tooltip 瞬间触发并紧密跟随指针，展示该节点的控制预算、实际确权、现金流和对应占比。
* 鼠标移入行时，左侧 label 的文字亮起，并平滑展现发光的 pastel 药丸状态标签（如 `用毕比 85%`）。

## 🚀 快速开始

请按照以下指令在本地运行并调试此原型项目：

### 1. 安装依赖
在项目根目录下安装所有 npm 依赖包：
```bash
cd dashboard
npm install
```

### 2. 启动开发服务器 (Webpack 模式)
使用 Webpack 引擎启动开发服务器（这可以有效规避因为 workspace 中包含中文路径可能触发的 Turbopack 编译报错）：
```bash
npm run dev
```

### 3. 生产环境构建验证
打包生成高度优化的静态分发生产包：
```bash
npm run build
```
